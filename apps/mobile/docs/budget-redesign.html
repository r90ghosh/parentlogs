Budget — same digest theme, 3 layouts

The budget has to make four things easy: browse & select items, toggle purchased + price tier (premium / best-value), see totals, and export. All three options below put a one-tap export in the header, use a checkbox to toggle purchased, and an Add control to select from the catalog. Same theme as the rest of the app.

Select — Add toggle on every catalog item Toggle — checkbox = purchased · tier switch Export — header action → Share / CSV / PDF Calm — one accent, no stat wall
